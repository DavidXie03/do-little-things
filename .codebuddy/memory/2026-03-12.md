# 2026-03-12

## Bug fixes - 4 issues in HomeView card animation and task management

1. **翻转动画卡背残留**：升起动画完成后隐藏 `risingCardRef` 的 `visibility`，防止翻转时多显示一个卡背。在翻转完成和安全超时后恢复。
2. **卡背显示逻辑**：改为基于 `totalRemainingCount`（总的未完成次数）而不是 `stackItems.length` 来判断是否显示卡背。新增 `backgroundCardCount` computed，用 `v-for="idx in backgroundCardCount"` 替代 `visibleStack.slice(1)`。
3. **新建每日任务今天不显示**：在 `addCustomAction` 中，如果今天的 `dailyTodos` 已生成且新任务今天应触发（`shouldTriggerOnDate`），则追加到今天的待办列表。
4. **删除任务后又出现在明天**：在 `removeTodoItem` 中同时删除对应的 `CustomAction`（通过 `task.id` 前缀 `custom_` 反查 `caId`），防止 regenerate 时重新创建。

Files modified: `HomeView.vue`, `useCustomActions.ts`, `useDailyTodos.ts`
