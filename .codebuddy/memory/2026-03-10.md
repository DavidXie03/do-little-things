# 2026-03-10

## UI/UX 改进修复
修复了5个界面问题：
1. **卡片切换动画** - 使用 Vue `<transition>` + CSS `@keyframes cardSwipeIn` 实现平滑的缩放+上移入场动画，替代原来突兀的直接出现
2. **卡片滑动 bug** - 在 `TaskCard.vue` 的 `animateOut` 中添加了 `oncancel` 回调和 `setTimeout` 安全保险，确保 `swipe` 事件一定会被触发
3. **标题位置** - "做件小事" 标题从卡片正上方移到页面顶部固定位置（pt-16），与卡片保持更大间距
4. **今日待办删除按钮** - 在 `useStorage.ts` 添加 `removeTodoItem` 函数，`PendingView.vue` 每个待办项右侧添加了删除按钮（IconTrash）
5. **弹窗样式优化** - 添加待办弹窗增加了内边距（px-8 py-7）、外 margin（20px）、输入框内边距（px-5 py-3.5）、按钮高度（py-3.5）
