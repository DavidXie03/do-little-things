## 修复日期中星期/月份显示为单字符的 bug

- **根因**: vue-i18n 的 `t()` 函数对数组类型的翻译值会返回逗号分隔的字符串，用数字下标访问会拿到单个字符（如 `e`、`.`、`t`）
- **修复**: 在 `PendingView.vue` 中将 `t('date.months')` / `t('date.weekdays')` 改为 `tm('date.months')` / `tm('date.weekdays')`
- `tm()` (translate message) 能返回原始数据结构（数组），确保下标访问正确
