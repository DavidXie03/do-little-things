# 2026-03-15

## 修复卡片滑动和主题色问题

完成 5 项改动：
1. **Toast 仅在全部完成后弹出** — `markTodoComplete` 改为返回 `boolean`，`HomeView` 中仅在返回 `true` 时弹 toast
2. **随机卡片遍历** — `loadStack` 使用 Fisher-Yates 洗牌 + 防连续重复
3. **滑动动画可中断** — 移除 `animPhase !== 'idle'` 拦截，新增 `interruptCurrentAnimation()` 方法
4. **Toast 颜色统一** — 改为使用 `--success` 绿色（与待办完成勾一致）
5. **主题色变量抽取** — 所有组件硬编码颜色（`#6C63FF`、`#00B894`、`#FDCB6E`）替换为 CSS 变量；新增 `--warning-zone`、`--success-light`、toast 配色变量，含深色模式覆盖

修改文件：`style.css`、`useDailyTodos.ts`、`useCardAnimation.ts`、`HomeView.vue`、`TaskCard.vue`、`AppToast.vue`、`TabBar.vue`、`TodoItem.vue`、`SettingsView.vue`

## 第三轮修改（4项）

1. **页面切换动画可打断** — `usePageSwipe.ts` 新增 `cancelAnimation()` 方法，`handleTouchStart` 和 `goToPage` 在动画进行中时打断当前动画而非忽略触摸
2. **卡片飞出动画回滚** — `useSwipeGesture.ts` 恢复原始 350ms duration 和 450ms 超时（移除之前的 200ms/120ms 提前回调优化）
3. **指示器恢复半圆** — `TaskCard.vue` 将全高渐变条改回半圆形状（`borderRadius: '0 100% 100% 0 / 0 50% 50% 0'`），居中于屏幕垂直方向
4. **扩展主题系统** — 从 2 种主题（紫色/蓝绿色）扩展为 5 种主题（梦幻紫/深海蓝/落日橙/森林绿/玫瑰红），每种主题有主色 + 辅色双色配置；主题选择从内联色块改为弹窗实现（`BaseModal` 组件），弹窗内显示双色圆点预览和主题名称

## 第四轮修改（3项）

1. **卡片翻转动画恢复** — 从 git 历史（ffd4d47 提交）恢复完整的卡片翻转动画系统：左滑时正面翻到90°→显示卡背→卡背缩小沉入牌堆→升起新卡背→翻转显示新正面；右滑时也有升起+翻转效果。`useCardAnimation.ts` 恢复了 AnimPhase 的 6 个阶段（idle/rising/flipping/front/left-flip/left-back/left-sink）和相应的 Web Animations API 动画
2. **半圆指示器改为平滑波浪** — `TaskCard.vue` 的左右滑动指示器从 CSS border-radius 半圆改为 SVG 贝塞尔曲线波浪形状，全屏高度，随拖拽进度动态变形，更加平滑自然
3. **辅助主题色应用** — `TodoItem.vue` 已完成勾选圈从固定绿色（`--success`）改为辅助主题色（`--secondary`）；`AppToast.vue` 的 success 类型 toast 颜色也改为基于 `--secondary`；`useTheme.ts` 新增每个主题的 `toastSuccessBg/Text` 属性
